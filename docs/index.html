Installing the plugin

To install the plugin you need three files.

  1. jQuery.js - the library itself
  2. notifyNote.jquery.js - main plugin script
  3. notifyNote.css - customizable styles for plugin

example:
 

<link rel="stylesheet" type="text/css" href="notifyNote.css">
<script src="jquery.js" type="text/javascript"></script>
<script src="jquery.notifyNote.js" type="text/javascript"></script>


 insert this to your <head> and locate files in same directory or specify path to them
 

Before using the plugin you need to initate it like this:
 

$('body').notifyNote()


 $('body') can be any element you want to add notifyNote
 

to apply some options to plugin you need the following construction
 

$('body').notifyNote({
   debug: true,
   option1: value,
   option2: value
})


 list options like this and you are ready to go further

  • debug

    Default: false

    Enables error logging.

  • horizontal

    Default: right

    Specifies a place for notification bar by horizontal. Possible values:

    1. left
    2. right

  • vertical

    Default: top

    Specifies a place for notification bar by vertical. Possible values:

    1. top
    2. bottom

  • id

    Default: notificationBar

    Allows to use a custom id for notificationBar.

  • width

    Default: 250px

    Specifies width for notifications.

  • max-height

    Default: 100px

    Specifies max-height for notifications.

  • z-index

    Default: 999

    Specifies z-index for notificationBar.

  • margin

    Default: 0

    Specifies margin for notificationBar.

  • prefix

    Default: notification_

    Specifies custom id prefix for notifications.

  • mouseDrag

    Default: true

    Enables closing notifications by dragging them from it's position to the opposite side. Possible values:

    1. true
    2. false

  • hideOnClick

    Default: true

    Enables closing notifications by clicking on them. Possible values:

    1. true
    2. false

  • closeButton

    Default: false

    Shows small close button in the right corner of notification.

  • replaceOnMatch

    Default: true

    If notification with same id already exists new notification will override old one.

    1. left
    2. right

  • beforeShow

    Event fires when notification is created, but not shown yet

  • afterShow

    Event fires right after notification is showed

  • afterShow

    Event fires right after notification is showed

  • beforeHide

    Event fires before notification hide

  • afterHide

    Event fires right after notification is hidden

  • beforeHideAll

    Event fires before all notifications hides

  • afterHideAll

    Event fires right after all notifications is hidden

  • textAdd

    Event fires after text added to notification

  • textReplace

    Event fires after notification text replaced

  • beforeDestroy

    Event fires before notificationBar destroys

  • notification(type, html, id)

    Simply shows notification.

    1. type - Type of notification. Default included: error, warning, info, darkGrey, lightGrey
    2. html - html content for notification, it can be anything from colored text to video frame
    3. id - notification id

    notifyNote.notification('error', 'I am notification!', 'test')

  • hideNotification(type, text, id)

    Hides already added notification.

    1. id - notification id

    notifyNote.notification('info', 'I gonna be hidden in 5 sec', 'hideMe')
    setTimeout(function(){
       notifyNote.hideNotification('hideMe')
    }, 5000)

  • hideNotifications()

    Hides all displayed notifications.

     

    notifyNote.notification('info', 'I gonna be hidden in 5 sec', 'hideMeAll')
    notifyNote.notification('warning', 'I gonna be hidden in 5 sec too', 'hideMeAll1')
    notifyNote.notification('darkGrey', 'Hide me hide me', 'hideMeAll2')
    notifyNote.notification('error', 'Not shure I have to say something', 'hideMeAll3')
    setTimeout(function(){
       notifyNote.hideNotifications()
    }, 5000)

  • addTextNotification(html, id)

    Adds html to specified notification.

    1. html - hrtml to append to notification
    2. id - notification id

    notifyNote.notification('warning', 'My text will be extended in 2 sec', 'addText')
    setTimeout(function(){
       notifyNote.addTextNotification('supertext', 'addText')
    }, 2000)

  • replaceTextNotification(text, id)

    Replaces html in specified notification.

    1. html - html to replace current notification html
    2. id - notification id

    notifyNote.notification('warning', 'My text will be replaced in 2 sec', 'replaceText')
    setTimeout(function(){
       notifyNote.replaceTextNotification('I am replaced', 'replaceText')
    }, 2000)

  • destroy()

    Destroys plugin and all notifications.

    notifyNote.notification('error', 'The end is close', 'destroy')
    setTimeout(function(){
       notifyNote.destroy()
    }, 2000)

  • Adding new notification types

    Creating your own notification style provided in 3 easy steps

    1. Write your css - add your style to notifyNote.css with class .notification.yourClass
    2. Call your notification - use your class name as type. example: notifyNote.notification('yourClass', 'New very own notification', 'own')
    3. There is no step 3.
